## Общие требования

- Python **3.9+**
- Доступ в интернет для установки зависимостей
- Рекомендуется собирать в чистом виртуальном окружении `**.venv`**
- Сборку выполняйте из **корня репозитория** (`.../ghostwriter`)

Команда PyInstaller **одинакова** на всех ОС (из корня проекта):

```bash
pyinstaller GhostWriter.spec --clean
```

Состав `datas` / `hiddenimports` и отличия BUNDLE на macOS описаны в [README.md](../README.md#сборка-pyinstaller) и в [ARCHITECTURE.md](ARCHITECTURE.md).

## Частые проблемы сборки

- `**ModuleNotFoundError` в runtime:**
  - удалите каталоги `build/` и `dist/`, повторите сборку с `**--clean`**;
  - проверьте, что сборка выполнена в активированном `.venv` с установленными зависимостями.
- Приложение запускается, но **нет записи:**
  - проверьте права на микрофон и параметр `audio_input_device` в `config/config.json`.
- **Не работает автовставка на macOS:**
  - проверьте **Accessibility** для процесса, от имени которого запущено приложение (см. [BUILD_macOS.md](BUILD_macOS.md)).

## Релизный чеклист

Перед публикацией сборки:

1. Очистить старые артефакты `build/` и `dist/`.
2. Выполнить свежую сборку на **целевой** ОС.
3. Проверить запуск и базовый сценарий диктовки (см. платформенный документ).
4. Проверить наличие нужных ресурсов в `dist` (включая `assets/models`, если используется локальный STT из бандла).
5. Зафиксировать версию и дату сборки в вашем процессе релиза.
